# Customer Dashboard Prototypes

These files are static HTML prototypes for customer-facing dashboard flows.
They are kept in `dashboard/public/customer-dashboard/` so they do not interfere with the existing React routes in `dashboard/src/pages/`.

## File placement summary

- `index.html`: full customer dashboard overview prototype
- `home.html`: alternate home dashboard prototype
- `login.html`: customer sign-in prototype
- `register.html`: customer sign-up prototype
- `watchlist.html`: customer watchlist prototype
- `buyer_profile.html`: buyer profile prototype
- `settings.html`: customer settings prototype
- `spend_analysis.html`: spend analytics prototype

## Extension mockup

- `extension/mockups/customer_extension_sidebar.html`: browser extension sidebar UI prototype

## How to preview

Start Vite in `dashboard/` and open:

- `/customer-dashboard/index.html`
- `/customer-dashboard/home.html`
- `/customer-dashboard/login.html`
- `/customer-dashboard/register.html`
- `/customer-dashboard/watchlist.html`
- `/customer-dashboard/buyer_profile.html`
- `/customer-dashboard/settings.html`
- `/customer-dashboard/spend_analysis.html`

These are static mockups and are not wired to backend APIs or React state.
